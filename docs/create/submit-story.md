### 3. Submit Your Story via GitHub Pull Request

**Step-by-Step PR Submission:**

1. **Fork the Repository**
   - Go to [https://github.com/suneeta-mall/fable-flow](https://github.com/suneeta-mall/fable-flow)
   - Click "Fork" to create your own copy

2. **Add Your Story**
   - Clone your forked repository
   - Create a new folder in `docs/books/` with your story name (use lowercase and underscores, e.g., `my_amazing_story`)
   - Copy all FableFlow-generated files into this folder
   - Ensure the structure matches the example in `docs/books/curious_cassie/curious_cassie_beach_ride_quests/`

3. **Create a Pull Request**
   - Commit your changes
   - Push to your fork
   - Create a pull request to the main repository
   - Use the PR template provided below

**Alternative: Share via GitHub Discussions**
If you're not comfortable with PRs, you can share your story in [GitHub Discussions](https://github.com/suneeta-mall/fable-flow/discussions) for community feedback first.
