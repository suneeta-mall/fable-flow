## üìù Content Guidelines

### Story Submissions
**Acceptable Content:**
- Original stories created with FableFlow
- Educational and entertaining narratives
- Age-appropriate content for specified audiences
- Stories that inspire curiosity and learning

**Please Avoid:**
- Copyrighted material without proper attribution
- Content that promotes harmful stereotypes
- Stories with inappropriate language or themes
- Purely promotional content without educational value

### Educational Standards
When creating educational content:
- Verify scientific accuracy
- Use reliable, credible sources
- Adapt complexity to target age group
- Include learning objectives when relevant

## üöÄ Getting Started in the Community

### New Members
1. **Introduce yourself** in GitHub Discussions
2. **Explore existing stories** to understand our style and quality
3. **Start small** - try our beginner tutorials first
4. **Ask questions** - we love helping new creators!

### Sharing Your Work
1. **Test thoroughly** before sharing
2. **Provide context** - explain your story's purpose and audience
3. **Include educational notes** if applicable
4. **Be open to feedback** and iteration

### Contributing Code
If you're interested in technical contributions:

1. Review our [technical documentation](../getting-started/installation.md)
2. Check existing issues and feature requests
3. Follow our coding standards and testing requirements
4. Submit pull requests with clear descriptions

### Respectful Communication
